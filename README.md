# Baseball-Pitching-Analysis-Python-Pybaseball
MLB Statcast pitching analysis using Python and Pybaseball to classify pitcher archetypes and evaluate relationships between fastball velocity, pitch usage, and performance metrics.
# MLB Pitching Performance Analysis

This project analyzes MLB pitching data using Statcast to explore pitcher tendencies and classify pitching styles based on fastball velocity and pitch usage.

Using the Pybaseball API, pitch-level Statcast data is extracted and processed through a full data analysis pipeline including cleaning, feature engineering, aggregation, and visualization.

Key metrics such as fastball velocity, pitch usage percentage, strikeout rate, hit rate, and run prevention indicators are calculated to evaluate pitcher performance and identify strategic pitching profiles.

Pitchers are grouped into archetypes such as **Power Fastball**, **Offspeed Heavy**, and **Balanced**, allowing for simplified interpretation of pitching strategies and their relationship to performance outcomes.
